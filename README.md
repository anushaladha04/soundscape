# cs35L-project

soundscape

env setup

1. clone repo
   git clone https://github.com/anushaladha04/soundscape.git
   cd soundscape
2. install dependencies
   cd client
   npm install
   cd ../server
   npm install
3. create environment files
   inside server folder, create a .env file with port info and MONGO_URI (ex)
4. start the backend
   cd server
   npm start
5. start the frontend
   cd client
   npm start
